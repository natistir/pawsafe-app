# Changelog

All notable changes to the PawSafe app will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-07-16

### Added
- Initial release of PawSafe app
- Home screen with paw safety guide and location input
- Location screen with surface temperature display
- Settings screen for user preferences
- GPS and zip code location support
- Surface temperature calculation algorithm
- Weather API integration ready
- Docker development environment
- TypeScript support
- React Native Paper UI components
- Expo framework integration

### Features
- Real-time surface temperature checking
- Color-coded safety risk levels (Safe/Caution/Dangerous/Extreme)
- Multiple surface type support (asphalt, concrete, grass, sand, metal)
- Location services with permission handling
- Comprehensive safety recommendations
- Professional Material Design UI

### Technical
- React Native 0.72.6
- Expo SDK 49
- TypeScript 5.1.3
- Docker containerized development
- Modular architecture with services and utilities
- Comprehensive type definitions

- Complete README with setup instructions
